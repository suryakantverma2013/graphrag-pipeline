# Widget W-9 Quick Start

The Widget W-9 is a compact relay module for the Orion control rack. This quick
start covers first power-up and basic wiring for the W-9 relay board.

## Wiring

Connect the 24 V supply to terminals A1 and A2. Land the control signal on
terminal S1 and the common return on terminal S2. Keep signal wiring separate
from the supply run to avoid coupling noise into the relay coil.

## First Power-Up

Apply power and confirm the status LED shows steady green. A blinking red LED
indicates a wiring fault on the supply terminals; remove power and recheck the
A1/A2 connections before retrying.
